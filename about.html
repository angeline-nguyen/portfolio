Where It All Started

The year is 2001. I have been introduced to a virtual pet site called Neopets. The website allowed users to customize various pages with something called HTML. Changing the appearance of the page with these tags felt almost magical. I was hooked.

I went to the library to borrow books on this magical HTML (replacing my previous obsession with non-fiction dog books). Soon, I was making my own little sites on Geocities. I remember the transition to needing to declare my doctype as "Strict". I remember laughing at memes about IE while simultaneously writing if clauses for if the user's browser was IE [some version]. I remember making layouts in tables, and I remember CSS becoming popular.

At the library, I picked up a book about CSS that introduced me to CSS Zen Garden. I also read a book by Jeffrey Zeldman about Web standards. I credit those two books for my current love of Web standards and for providing the foundation for my enthusiasm for Web accessibility.

Where It's Going

I am currently enrolled in a professional certificate in coding program for full stack development with MERN. My goal is to become a Web developer and eventually specialize in Web accessibility. If you know someone or are someone who is in the Web accessibility field, I would love to speak with you. Bonus points if you're interested in mentoring a motivated individual (me) on her journey.
